# Sparse Cashflow Value Alpha Proposal

> [!abstract] Research Thesis
> **Sparse Cashflow Value** ranks firms by enterprise value relative to operating cash flow, but only trades when the valuation signal is extreme. The thesis is that cash-flow valuation contains slow-moving information, while sparse activation reduces noise from moderate observations.

| Activation | `rank(abs(y)) > 0.90` |

| Main risk | Value trap and accounting denominator instability |

## Formula

```text
x = winsorize(ts_backfill(enterprise_value/cashflow_op, 120), std=4);
y = ts_mean(x, 30);
trade_when(rank(abs(y)) > 0.90, group_rank(y, industry), -1)
```

## Economic Rationale

Enterprise value to operating cash flow is a valuation ratio. It asks how expensive the firm is relative to cash generated by operations. The sparse gate avoids turning every moderate valuation difference into a position and instead concentrates the book in stronger valuation dislocations.

This signal is useful because it is simple and not obviously a turnover or options-skew variant. Its low turnover is also consistent with a valuation/accounting signal.
